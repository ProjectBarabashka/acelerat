## 🇺🇸 English Version

> **Status: ⚠️ BETA STAGE (2026)**  
> The project is currently in active development. Features and interface are being improved.

A professional tool for monitoring and accelerating "stuck" Bitcoin transactions. T_U_R_B_O_T_X provides a powerful interface to manage your transactions when the network is congested.

### 🔥 Key Features:
*   **🌍 Universal Localization** — The interface supports instant translation into any language.
*   **⛓ Advanced Acceleration** — Built-in support for **CPFP** (Child Pays For Parent) and **RBF** (Replace-By-Fee) logic.
*   **📊 Smart Status Indicators** — Real-time tracking with intuitive visual progress bars showing the exact stage in the mempool.
*   **📱 Responsive Design** — Fully optimized for both desktop and mobile devices.

### 💳 Service Details & Limits:
*   **🎁 Free Community Limit:** 100 free accelerations per day for all users total.
*   **💳 One-Time Payment:** Individual paid acceleration is available after the daily free limit is reached.
*   **⚡️ Priority:** Paid transactions receive maximum priority in the acceleration queue.
